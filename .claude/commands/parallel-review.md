# 병렬 코드 리뷰 (Multi-Agent)

4개의 전문 에이전트가 병렬로 코드를 리뷰합니다.

## 워크플로우

```
┌─────────────────────────────────────────┐
│           Supervisor Agent              │
│       (리뷰 범위 분석 및 할당)           │
└────────────────┬────────────────────────┘
                 │
    ┌────────────┼────────────┬────────────┐
    ▼            ▼            ▼            ▼
┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐
│Security│  │ Logic  │  │ Style  │  │  Perf  │
│Reviewer│  │Reviewer│  │Reviewer│  │Reviewer│
└────┬───┘  └────┬───┘  └────┬───┘  └────┬───┘
     │           │           │           │
     └───────────┴───────────┴───────────┘
                 │
                 ▼
         ┌───────────────┐
         │  Aggregator   │
         │ (종합 리포트)  │
         └───────────────┘
```

## 리뷰어 역할

| 에이전트 | 역할 | 검토 항목 |
|----------|------|-----------|
| **Security Reviewer** | 보안 취약점 분석 | SQL Injection, XSS, CSRF, 인증/인가, 민감 데이터 노출 |
| **Logic Reviewer** | 로직 정확성 검토 | 알고리즘, 엣지 케이스, 에러 처리, 비즈니스 로직 |
| **Style Reviewer** | 코드 스타일 검토 | 명명 규칙, 포맷팅, 주석, 가독성, 일관성 |
| **Performance Reviewer** | 성능 분석 | 시간 복잡도, 메모리 사용, N+1 쿼리, 캐싱 기회 |

## 실행 방법

이 커맨드가 실행되면:

1. **코드 범위 확인**: 리뷰할 파일/디렉토리 지정
2. **병렬 분석**: 4개 전문 리뷰어가 동시에 분석
3. **이슈 수집**: 각 리뷰어의 발견 사항 취합
4. **우선순위 지정**: 심각도에 따라 이슈 분류
5. **종합 리포트**: 통합 리뷰 보고서 생성

## 출력 형식

```markdown
# 코드 리뷰 보고서

## 요약
- **파일 수**: N개
- **발견된 이슈**: X개 (Critical: A, High: B, Medium: C, Low: D)
- **전체 점수**: XX/100

## 🔴 Critical Issues
### [파일명:라인] 이슈 제목
**유형**: Security | Logic | Performance
**설명**: ...
**권장 수정**:
```diff
- 문제 코드
+ 수정 코드
```

## 🟠 High Priority
...

## 🟡 Medium Priority
...

## 🟢 Low Priority / Suggestions
...

## 리뷰어별 상세 분석

### Security Review
...

### Logic Review
...

### Style Review
...

### Performance Review
...

## 권장사항
1. [즉시 수정 필요 사항]
2. [개선 권장 사항]
3. [장기적 고려 사항]
```

## 심각도 기준

| 레벨 | 설명 | 예시 |
|------|------|------|
| **Critical** | 즉시 수정 필요, 보안/데이터 위험 | SQL Injection, 인증 우회 |
| **High** | 빠른 수정 필요, 기능 오류 | 무한 루프, 메모리 누수 |
| **Medium** | 수정 권장, 유지보수 이슈 | 중복 코드, 복잡한 로직 |
| **Low** | 개선 제안, 스타일 이슈 | 명명 규칙, 주석 부족 |
