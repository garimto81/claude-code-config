# 솔루션 검색

웹과 GitHub를 검색하여 최신 솔루션을 찾아 추천합니다.

## 입력 정보

다음 정보를 사용자에게 질문하세요:

1. **검색 키워드**: 찾고자 하는 기술/문제
2. **기술 스택**: 관련 언어/프레임워크 (선택)
3. **용도**: 학습 | 프로덕션 | 사이드 프로젝트 (선택)

## 검색 프로세스

### Step 1: 웹 검색 (WebSearch)

다음 소스에서 검색:
- 공식 문서
- Stack Overflow
- Dev.to / Medium 기술 블로그
- Reddit (r/programming, r/webdev 등)
- Hacker News

검색 쿼리 예시:
- `[키워드] best practices 2024`
- `[키워드] vs [대안] comparison`
- `[키워드] tutorial production`

### Step 2: GitHub 검색

```bash
# 관련 저장소 검색
gh search repos "[키워드]" --sort stars --limit 10

# 최근 업데이트된 프로젝트
gh search repos "[키워드]" --sort updated --limit 5
```

검색 기준:
- ⭐ Stars 수 (인기도)
- 최근 커밋 날짜 (활성도)
- Issue 해결률
- 문서화 수준

### Step 3: 결과 정리
